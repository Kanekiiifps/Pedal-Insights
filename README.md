# NYC Citi Bike Trip Analysis – October 1st, 2013

Exploratory data analysis of **Citi Bike trips** in New York City on October 1st, 2013.

This notebook answers key questions such as:
- How does trip duration vary by time of day?
- What percentage of riders are annual subscribers?
- Which bike stations are the most popular (most trip starts)?

Interactive map included using **Folium**!

## Key Findings

| Question | Insight |
|---|--------|
| **Subscribers** | **87.45%** of riders on Oct 1, 2013 were annual subscribers |
| **Peak Hours** | Most trips start between 7–9 AM and 5–7 PM (classic commuting pattern) |
| **Most Popular Station** | Pershing Square North – over 400 trips started here in one day! |
| **Trip Duration** | Short spikes throughout the day, longer recreational trips around midday |
